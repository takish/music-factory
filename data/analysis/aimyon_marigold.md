# 曲分析: マリーゴールド / あいみょん

## 曲の本質

- **懐かしさと新鮮さの両立**: カノン進行（80年代から日本人に馴染み深い）を基盤にしながら、16分音符を織り交ぜたリズムで古臭さを回避。「どこかで聴いたことがある安心感」と「今の音」が共存
- **滑らかに下降するベースライン**: オンコード（分数コード）を多用し、ベース音が緩やかに下降。これが曲全体に流れるような穏やかさと切なさを与える
- **明暗の同居**: メロディは「明るい部分では主要な音を回避し、暗い部分では確実に響かせる」設計。爽やかながら物悲しい、相反する感情が一曲に溶け込む

## Harmony / Chord Progression（機能和声）

**Key**: D Major

### Intro / Verse（Aメロ）
- **パターン**: I → V/VII → vi → iii → IV → I/III → IV → V
- **実コード**: D → A/C# → Bm → F#m → G → D/F# → G → A
- **雰囲気**: カノン進行の変形。オンコードによりベースがD→C#→B→F#→G→F#→G→Aと滑らかに動く
- **設計意図**: 日本人に刷り込まれた「聴き覚えのある安心感」を提供しつつ、ベースの動きで退屈さを回避

### Pre-Chorus（Bメロ）
- **パターン**: vi → ... → V
- **実コード**: Bm で開始 → A で終止
- **雰囲気**: マイナー開始で一時的に暗転。並行調Bmへの仮転調
- **設計意図**: サビ前に「沈む」ことでサビの浮上感を強調。ドミナント（A）で次への期待を高める

### Chorus（サビ）
- **パターン**: I → V/VII → vi → iii → IV → I/III → vi → V
- **実コード**: D → A/C# → Bm → F#m → G → D/F# → Bm → A
- **雰囲気**: Aメロとほぼ同型だが、6小節目で D/F# → Bm の強進行
- **設計意図**: コードを変えずメロディで勝負。強進行で推進力を加え「サビ感」を演出

## Arrangement

- **ジャンル**: J-Pop × フォーク × ギターポップ
- **特徴**: アコースティックギター中心のシンプルなバンドサウンド。弾き語りでも成立する構成
- **設計**: 音数を絞り「歌」を主役に。装飾を抑え、言葉とメロディの力で聴かせる
- **Drop**: なし（エレクトロニック要素なし）

### 密度設計

- **Intro**: アコギアルペジオのみ。最小構成
- **Verse**: アコギ + ドラム + ベース。控えめ
- **Pre-Chorus**: 同上。沈む
- **Chorus**: フルバンド + ストリングス。しかし音数は抑制的
- **Final Chorus**: 同上。派手にしすぎない

### リズム / プロダクション設計

- **基調リズム**: 8ビート（BPM 106のミドルテンポ）
- **ハイハット**: 8分刻み基調。16分のゴーストノートが要所で躍動感を加える
- **ベースタイプ**: 生ベース（ルート中心、オンコードに追従）
- **影響元**: 90年代〜00年代J-Pop / フォーク / ギターロック
- **特徴的手法**: オンコード（分数コード）によるベースの滑らかな下降

## Lyrics Design

- **言語**: 日本語
- **視点**: 一人称（私）
- **主題**: 夏の恋の記憶、過ぎ去った時間への切なさ、花に託した感情
- **言語密度**: 中（ゆったりしたテンポに合わせた言葉の置き方）
- **断定表現**: 少なめ。問いかけや回想が多い
- **情景描写**: 多い（風、夏、花、空など自然の描写）
- **感情**: 語りすぎず、情景に感情を溶かす。「悲しい」と言わず悲しさを伝える

## 設計のポイント

この曲から学べる、再利用可能なテクニック：

- **カノン進行 + オンコード**: 王道進行に分数コードを加えることで「聴き覚えはあるが新鮮」を実現
- **Bメロのマイナー転調**: 並行調（vi）への一時転調でサビ前に「沈める」。浮上のコントラストを強調
- **リズムの複雑性で古臭さ回避**: 16分音符のゴーストノートで懐かしいコード進行を現代的に
- **音数の抑制**: 派手にせず「歌」を主役に。弾き語りでも成立する強度
- **情景で感情を語る**: 直接的な感情表現を避け、自然描写に感情を託す
